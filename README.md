Fork de [dungeon-revealer](https://github.com/dungeon-revealer/dungeon-revealer), adapté pour les besoins de ma campagne Donjons & Dragons.

## Fonctionnalités ajoutées

- **Cartes YouTube** : possibilité d'utiliser une URL YouTube comme carte (lecture en boucle, sans son, sans contrôles visibles)
- **Brouillard de guerre animé** : shader GLSL animé appliqué au brouillard pour un rendu plus immersif
- **Interface en français** : traduction de l'ensemble de l'interface utilisateur
- **Nouveaux visuels** : fond d'écran de connexion et icônes personnalisés

## Fonctionnalités de base (dungeon-revealer)

- Révélation progressive de la carte aux joueurs via le brouillard de guerre
- Déplacement de tokens sur la carte
- Chat avec lancers de dés intégrés
- Partage de notes et d'images entre le MJ et les joueurs
- Compatible desktop et mobile (tablettes, téléphones)
- Utilisable en local ou via internet

## Lancer le projet

```bash
npm install
npm run dev
```
